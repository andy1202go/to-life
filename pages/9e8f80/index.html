JavaScript入门书籍和课程推荐

如果想入门学习 JavaScript，我主要推荐以下在线教程。

  • MDN JavaScript 教程 (opens new window)，你可以认为这是最权威的 JavaScript 官方教程了，从初级到中级再到高级。
  • W3School JavaScript (opens new window) 教程，这个教程比较偏 Web 方面的编程。
  • JavaScript 全栈教程（廖雪峰） (opens new window)，这是廖雪峰的一个比较偏应用的教程，也是偏 Web 方面的编程，同时包括涉及后端的 Node.js 方面的教程。
